## 요약

온보딩 2·3단계에 **← 이전** 네비게이션을 추가하고, 텔레그램 ON 시 3단계가 이어진다는 것을 스텝per·버튼 문구·안내로 명확히 했다. 가입 완료 후 3→2 이동 시 알림 설정만 수정하고 다시 연동 단계로 돌아갈 수 있다.

## 흐름 (텔레그램 ON)

1. 관심 종목 → 2. 알림 설정 → **다음 단계** (가입) → 3. 텔레그램 연동  
2. 3단계 **← 알림 설정** → 설정 수정 → **텔레그램 연동 계속**  
3. **나중에 하기** / **완료** → 미연동 시 알림 OFF 동기화 후 홈
