# MarketLens UI · 제품 화면 개요

**목적:** 코드·PR만 보기 어려운 **현재 제품의 전반적인 화면 톤·패턴**을 한곳에 정리한다.  
**구현 근거:** 색·간격·타이포 등 수치는 `src/styles/tokens.css` 및 [디자인 토큰 changelog](../changelog/2026-05-15-feat-design-tokens.md)를 따른다.

## 1. 공통 셸 (앱 뼈대)

### 상단

- **브랜드:** 좌측 `MarketLens` 로고 + 워드마크.
- **글로벌 내비:** `홈` · `종목` · `언급량 급등` · `인물 발언` — 현재 라우트는 배경·테두리로 **활성 탭** 강조.
- **우측 유틸:** `관심 목록`, 검색창, **설정(톱니)** — 설정 클릭 시 드롭다운에서 **언어 / 통화**, `마이페이지`, `로그아웃` 등 (테마 선택 UI 없음).

### 티커 바

- 메인 내비 아래 **가로 스크롤 티커** — 지수·종목 요약, 등락은 **녹색(상승·긍정)** / **적색(하락·부정)** 계열로 구분.

### 본문

- 대부분의 페이지는 **`Layout`** 안에서 렌더되며, 본문은 **`main[data-scroll-root]`** 기준 세로 스크롤.
- **카드형 UI:** 섹션 단위로 `Card` 또는 동등한 서피스(`--color-bg-card`, `--color-bg-section`) + **둥근 모서리**(`--radius-*`).

### 감성·수치 색 규칙 (의미)

- **긍정 / 상승 / 호재:** 녹색 계열 (`--color-success` 및 토큰 믹스).
- **부정 / 하락 / 경고:** 적색 계열 (`--color-danger` 등).
- **중립·참고:** 회색·노랑 구간(게이지 중앙대 등).
- **주요 액션:** 파란 **프라이머리** (`--color-primary`, 인터랙티브 텍스트는 `--color-interactive-active-text`).
- **클릭 가능 행·카드:** 호버·포커스는 [interactive-surfaces.md](./interactive-surfaces.md) 패턴(A 링 행 / B 테이블 / C 텍스트만 / GNB 예외)을 따른다. 수치 토큰은 `--interactive-*` (`tokens.css`).

## 2. 홈 (`/`)

**역할:** 포트폴리오·시장 맥락을 한 화면에서 스캔.

| 영역 | 설명 |
|------|------|
| **헤더** | `PageHeader` — 제목 `홈`, 부제로 화면 목적 안내. |
| **좌 열** | **내 포트폴리오 감성** — 반원형 게이지(스케일 음수~양수), 바늘/점 + 라벨(예: 중립). **언급량 급증 TOP3** — 종목명 + 변화율(색상 강조). **KOSPI 종합** — 동일 게이지 패턴으로 시장 감성. |
| **우 열** | **내 관심 종목 워치리스트** — Grid(종목·현재가·등락·감성·뉴스·언급률). 행 단위 클릭 시 **`/stock/:code`** 이동. |
| **하단** | **KOSPI + 섹터 감성 히트맵** — Treemap으로 섹터별 면적·색(감성 점수). |

**레이아웃:** 상단 2열(좌 게이지·TOP3 / 우 워치리스트) + 하단 시장 블록. 좌측 열 높이에 맞춰 우측 목록 **내부만 스크롤**하는 패턴.

---

## 3. 종목 상세 (`/stock/:code`)

**역할:** 단일 종목의 감성·뉴스·연관 정보를 깊게 탐색.

| 영역 | 설명 |
|------|------|
| **헤더** | 종목명(굵게), 티커·시장·섹터 메타. **감성 점수**·**언급량 변화율** 배지. **오늘 핫 이슈** 요약 문단. **관심종목** 토글. |
| **중앙 좌** | **30일 감성 추이** — Lightweight Charts 스타일 라인, Y축 구간(극도 긍정~극도 부정) 배경색, 하단 **언급량 막대**. |
| **중앙 우** | **감성 분류 분포(오늘)** — 가로 스택 바 + 범례(긍정/중립/부정 건수). **30일 평균 대비 현재 위치** — 현재·평균·최고 박스 비교. |
| **하단 좌** | **관련 뉴스** — 필터(전체/긍정/부정). 리스트: 시각·매체·감성 점수·헤드라인·스니펫·썸네일. **무한 스크롤** + 하단 **맨 위로** 플로팅. |
| **하단 우 (사이드)** | **연관 종목** 리스트(감성·등락 색). **인물 발언 타임라인** — 인물·소속·시간·요약·감성 점수. |

**스크롤:** 라우트 변경 시 메인 스크롤 상단 리셋. 우측 컬럼은 **sticky** + 뉴스 영역과 시각적 균형.

---

## 4. 언급량 급등 (`/buzz`)

**역할:** 전 시장에서 언급이 급증한 종목 랭킹.

- **좌 상단 카드 3종:** 최대 급등 종목·평균 감성·오늘 뉴스 건수 등 **요약 지표**.
- **우 메인:** **언급량 급등 TOP 10** 테이블 — 순위, 종목, 오늘 건수, 급등률(적색 강조), 감성 점수(녹/적), **AI 요약(왜?)** 컬럼으로 한 줄 설명.

**톤:** 다크 카드 + 테이블 가독성, 숫자·퍼센트는 색으로 즉시 읽히게.

---

## 5. 인물 발언 (`/person`)

**역할:** 주요 인물 발언 피드 + 기간별 화제 인물·자주 언급 종목.

| 영역 | 설명 |
|------|------|
| **좌 메인** | 발언 **카드 리스트** — 아바타(이니셜), 이름·소속·출처·상대 시각, **긍정/부정 배지와 점수**, 본문 인용/요약 블록. |
| **우 사이드** | **오늘 / 7일** 세그먼트 토글(API `range`). **언급량 TOP 5 인물**. **자주 언급된 종목** — `#종목명 N` 칩 그리드. |
| **스크롤** | 우측 패널만 **내부 스크롤**; **맨 위로**는 `BackToTopButton` + `scrollRootSelector`로 해당 영역 기준. |
| **피드** | **무한 스크롤**(커서 페이지네이션). |

---

## 6. 마이페이지 (`/mypage`)

**역할:** 관심 종목·알림·계정 요약.

- **상단 요약 카드:** 관심 종목 수, 긍정 비율, 주의 필요(감성 임계) 등.
- **관심 종목 리스트:** 표 + **추가** 버튼, 행별 알림·삭제.
- **알림 설정:** 토글 리스트(언급량 급등, 감성 급락, 인물 발언, 일간 요약 등) + **알림 예시** 박스.
- **계정 정보:** 닉네임, 이메일, 가입일, 플랜.

**토글:** ON 시 파란 트랙 등 **인터랙티브 토큰**과 맞춘 스타일.

## 7. 에러·빈 상태 (공통 UX)

- **일반 fetch 실패:** `PageFetchError` — 레이아웃 안 **embedded 카드**(제목 + 메시지 + 홈/이전).
- **HTTP 401·403·404·5xx·네트워크:** `AppErrorPage` **fullscreen** — 별 배경·큰 블러 상태 코드·프리셋 문구·CTA (`/dev/errors/:variant`와 동일 톤).
- 디자인 토큰 기반 색·타이포 — [AppErrorPage 토큰화 기록](../changelog/2026-05-20-feat-core-pages-frontend.md) 참고.

## 8. 스냅샷 권장 (선택)

PR·DDR 근거로 쓰기 좋은 화면 단위:

1. 홈 — 2열 + 하단 히트맵  
2. 종목 상세 — 헤더 + 차트 + 뉴스 + 사이드  
3. 언급량 급등 — 요약 카드 + TOP10  
4. 인물 발언 — 피드 + 우측 기간·TOP5·칩  
5. 마이페이지 — 요약 + 표 + 알림 + 계정  
6. 설정 드롭다운 열림 상태  

파일은 `docs/snapshots/YYYY-MM-DD/` 아래에 두고, [snapshots/README.md](../snapshots/README.md)에 한 줄 인덱스를 추가하는 방식을 권장한다.
